Linux



Bem-vindo ao nosso guia prático para os principais comandos básicos do Linux! Vamos explorar o maravilhoso mundo do sistema operacional Linux e aprender como utilizar os comandos fundamentais para navegar, gerenciar arquivos e executar tarefas no terminal. O Linux é um sistema operacional de código aberto amplamente utilizado em servidores, dispositivos embarcados e até mesmo nos computadores pessoais. Sua flexibilidade, estabilidade e poder de personalização o tornaram uma escolha popular entre os usuários de tecnologia. No entanto, para aproveitar ao máximo o Linux, é essencial entender os comandos básicos do terminal. Durante este tutorial, você aprenderá comandos essenciais como ls, cd, mkdir, rm e muitos outros, que permitirão a você navegar pelo sistema de arquivos, criar e excluir diretórios, copiar e mover arquivos, entre outras tarefas importantes. Além disso, vamos explorar alguns recursos adicionais, como redirecionamento de entrada e saída, gerenciamento de permissões de arquivo e como obter ajuda quando precisar. Independentemente de você ser um iniciante absoluto no mundo do Linux ou já ter alguma experiência, este guia oferecerá uma base sólida para ajudá-lo a se tornar mais confortável com o uso do terminal e a aproveitar ao máximo os recursos do Linux. Então, vamos começar a mergulhar nesse universo poderoso e emocionante do Linux, dominando os principais comandos básicos que abrirão as portas para um novo nível de produtividade e conhecimento!



Confira abaixo uma lista com os 9 comandos mais usados nos sistemas operacionais baseados em Linux:


  1. pwd: encontra o caminho completo do diretório atual.

  2. cd: permite navegar até determinada pasta.

  3. ls: lista todos os arquivos e pastas dentro de um diretório.

  4. cat: lista os conteúdos de um arquivo de texto na saída padrão (sdout).

  5. cp: copia arquivos do diretório atual para uma pasta diferente.

  6. mv: pode ser usado para mover ou renomear arquivos.

  7. mkdir: cria um novo diretório.

  8. rm: remove arquivos e diretórios.

  9. sudo: executa um comando como superusuário.


1. Comando pwd


Use o comando pwd para encontrar o caminho para o diretório atual (da pasta) em que você está. O comando vai retornar um caminho completo (cheio), que é basicamente um caminho que começa com uma barra inclinada (/). Um exemplo de um caminho completo é /home/username.


O comando pwd usa a seguinte sintaxe:


pwd [opção]


Ele tem a opção aceitável:


-L ou -logical imprime o conteúdo da variável de ambiente, incluindo links simbólicos.
